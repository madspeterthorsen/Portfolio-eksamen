Tema 2 - Grundlæggende web

I dette projekt arbejdede vi med at designe og kode vores første komplette website. Vi fik udleveret billeder, tekst, wireframe og layoutdiagram, og skulle derefter bygge et website fra bunden. Vi startede med at kode en mobilversion (mobile first) og videreudviklede den til desktop. Undervejs arbejdede vi med semantisk HTML, CSS Grid & Flexbox, media queries og responsivt design. Det var også første gang vi skulle strukturere et webprojekt korrekt og uploade det til vores eget domæne.

Jeg har lært:

  • Strukturere HTML korrekt og semantisk
  • Bruge CSS til layout (Grid, Flexbox, margin og padding)
  • Brug af Figma til wireframes og Præsentationskort
  • Arbejde med mobile first og responsivt design (media queries)
  • Opsætte korrekt filstruktur og uploade til domæne
